Charge users a small fee to access your content, using the Pica-Pay micro payments platform.

== Description ==
Integrate the Pica-Pay micro payments platform into your website.  This enables you to charge users a small fee to access individual content on your website.

Sign up with Pica-Pay for free at https://www.pica-pay.com, install this plugin and start charging users to access your content.

== Installation ==
1. Upload the Pica-Pay plugin to the `/wp-content/plugins/` directory
2. Activate it through the 'Plugins' menu in WordPress
3. Configure it on the settings page (under the admin 'Settings' menu)
4. On each page or post for which you wish to charge, set the Pica-Pay 'paid' toggle on, and enter the charge amount.
